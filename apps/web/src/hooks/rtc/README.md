# RTC Hooks

This folder contains the LiveKit voice runtime and shared RTC-related types/config used by the web app.

- `useLivekitVoiceRuntime.ts` - main LiveKit connect/disconnect and track lifecycle hook.
- `voiceCallConfig.ts` - RTC constants and thresholds.
- `voiceCallTypes.ts` - shared payload and UI state types used by the voice runtime.
